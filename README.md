# Web Zen

> **Warning:** The entirety of this repository (including this README) was shamelessly generated by AI.

A Chrome/Edge extension that helps you stay focused by limiting visits to distracting websites. This is a bootleg implementation of the iOS ScreenZen app.
Helps to take care of that FOMO and social media addiction illness (allegedly).

## Installation

I couldn't be bothered to upload this nonsense to the extension store, so:
1. Clone or download this repository.
2. Open `chrome://extensions` in Chrome/Edge.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the project folder.

## How It Works

1. **Add websites** — Open the extension popup and list the hostnames you want to limit (e.g. `reddit.com`, `youtube.com`). Be sure to list just the hostnames, I've no idea if trailing slashes work and don't care if they don't.
2. **Set a budget** — Choose how many sessions per day you're allowed and how long each session lasts.
3. **Stay intentional** — When you navigate to a tracked site, Web Zen intercepts the page and asks *"Is this important?"*. You must wait 10 seconds before you can proceed, giving you a moment to reconsider. I also didn't care to make this waiting threshold configurable. Use AI to do it yourself.
4. **Sessions run out** — Once you've used all your sessions for the day, the site stays blocked until midnight.

## Features

- **Configurable site list** — Track any number of hostnames.
- **Session limits** — Set 1–100 sessions per day with durations from 10 seconds to 30 minutes.
- **Custom confirmation message** — Personalise the prompt shown when you visit a tracked site (very useful).
- **Streak tracking** — See how many consecutive days you've stayed within your limits.
- **Daily auto-reset** — Usage counters reset at midnight automatically.

## Usage

Click the Web Zen icon in the toolbar to open the popup, configure your tracked sites and session settings, then click **Save settings**.
